# Aula 06 - Próximos passos da feature de equipamentos

Esta aula deve continuar a partir da base visual criada na Aula 05.

## Possíveis objetivos

- Fazer os filtros alterarem a lista da tabela.
- Criar o formulário de novo equipamento.
- Transformar as ações da tabela em ações reais.
- Criar a tela de detalhes do equipamento.
- Preparar uma simulação de chamada de API.
- Começar a discutir backend, autenticação e persistência.

## Pontos que ficaram fora da Aula 05

- Os filtros da Aula 05 são visuais.
- O botão "Novo equipamento" ainda não abre formulário.
- As ações da tabela ainda mostram apenas uma mensagem simples.
- A tela de detalhes ainda não existe.
- Os dados ainda são mockados.

## Sugestão de ordem

1. Implementar filtro por texto.
2. Implementar filtro por status.
3. Implementar filtro por tipo.
4. Criar estado para abrir e fechar modal.
5. Criar formulário simples com Ant Design.
6. Conectar o botão "Novo equipamento" ao modal.
7. Preparar a rota de detalhes.
